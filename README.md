# 后天八卦 · 静心图 🌀

一个基于 Electron 的桌面悬浮小挂件，展示标准的**后天八卦图**。点击旋转，用于静心、凝神、调节情绪。

---

## ✨ 缘起

这不是一个普通的八卦图，而是一份写给自己的"清静经"。

人生总有烦闷之时。此项目的初衷，是打造一个**数字时代的安心法器**——当你感到焦躁、思绪纷飞时，打开它，点击它，看着八卦缓缓流转，心神便随之安定。

> **万物负阴而抱阳，冲气以为和，吾心清静。**
> ——《道德经》

## ☯ 文化内涵

### 后天八卦

本图采用**文王后天八卦**的方位排列，象征万物在时空中的运行规律：

| 方位 | 卦名 | 符号 | 象征 | 意涵 |
|:---|:---|:---|:---|:---|
| 正北（下） | 坎 | ☵ | 水 | 润下，险陷中藏生机 |
| 正南（上） | 离 | ☲ | 火 | 炎上，光明依附柔顺 |
| 正东（左） | 震 | ☳ | 雷 | 震动，惊蛰而后奋起 |
| 正西（右） | 兑 | ☱ | 泽 | 悦纳，柔外刚中 |
| 东南（左上）| 巽 | ☴ | 风 | 入微，柔顺而能渗透 |
| 西南（右上）| 坤 | ☷ | 地 | 厚德，承载包容万物 |
| 东北（左下）| 艮 | ☶ | 山 | 静止，知止而后有定 |
| 西北（右下）| 乾 | ☰ | 天 | 刚健，自强而不息 |

### 阴阳鱼

图中央的阴阳鱼，黑中有白，白中有黑——**"阴中有阳，阳中有阴"**。逆境中藏着转机，顺境中需要冷静。两个鱼眼取石青色，为"水"的点睛之笔——**水利万物而不争**。

### 五行补水

配色刻意偏冷，深蓝、赭石、石青……因为**水主智，主静**。当你感到烦躁（火旺），需要"补水"以降火安神。坎卦（☵）以冰蓝色稍作突出，正是此意。

## 🎨 视觉风格

整体采用**古画宣纸**风格，追求"拙朴"的美学意境：

- **背景**：宣纸暖灰 `#F2EFE9`，如古书扉页
- **阴阳鱼**：石墨黑 `#3A3C3B` / 牙白 `#FDFBF7`，水墨般柔和
- **卦象**：赭石色 `#6B4E3D`，似古印泥
- **外圈**：古铜色 `#8B7D6B`，如铜镜质感
- **坎卦与鱼眼**：石青色 `#5B7B8A`，唯一冷色，象征水的清凉

无阴影、无渐变、无发光——**扁平如印在纸上**。

## 🖱 交互方式

| 操作 | 效果 | 场景 |
|------|------|------|
| **单击** | 慢转 ↔ 停止（12 秒一圈） | 日常烦躁时盯着看，平复心绪 |
| **双击** | 快转 ↔ 停止（2 秒一圈） | 心绪纷飞时快速"甩出"杂念 |
| **长按** | 按住越久转越快，无上限 | 极度憋闷时用，观想烦闷被甩出消散 |
| **Ctrl + 滚轮** | 缩放大小（40% ~ 150%） | 调节到最舒服的大小 |
| **拖拽** | 移动位置 | 放到屏幕角落默默陪你 |
| **右键** | 关闭 | — |

## 🔧 安装与运行

### 环境要求

- [Node.js](https://nodejs.org) ≥ 18
- Windows 10/11（macOS 也可运行）

### 步骤

```bash
# 1. 克隆
git clone https://github.com/BIGGG-BOT/bagua-widget.git
cd bagua-widget

# 2. 安装依赖
npm install

# 3. 启动
node_modules\electron\dist\electron.exe .
```

Windows 用户也可直接双击 `启动八卦.bat`。

### Electron 下载失败（国内网络）

```bash
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install
```

## 🛠 自定义

在 `renderer.js` 中调整：

| 常量 | 说明 | 默认值 |
|------|------|--------|
| `currentScale` | 默认大小 | `0.75` |
| `SPEED_SLOW` | 慢转速度（度/秒） | `30` |
| `SPEED_FAST` | 快转速度（度/秒） | `180` |
| `CHARGE_RATE` | 长按加速率（度/秒²） | `120` |

配色直接搜索颜色代码替换即可。
