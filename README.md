# 后天八卦 · 静心图 🌀

桌面上一个小小的八卦挂件。点一下，它就转起来，安安静静陪着你。

---

## 🕯️ 缘起

这不是一个普通的八卦图，而是一份写给自己的"清静经"。

人生总有烦闷的时候。这个项目的初衷很小——做一个**数字时代的安心小物**。焦躁的时候、思绪纷飞的时候，打开它，点一下，看着它缓缓地转，心就慢慢沉下来了。

> **万物负阴而抱阳，冲气以为和，吾心清静。**
> ——《道德经》

## ☯️ 文化内涵

### 后天八卦

这里用的是**文王后天八卦**的排法。不同于伏羲先天八卦的"天地定位"，后天八卦画的是**万物在时间里流转变迁**——春天发芽、夏天疯长、秋天结果、冬天休息，一天的晨昏、一生的起落，都在这张图里。

| 方位 | 卦名 | 符号 | 卦象 | 意涵 |
|:---|:---|:---|:---|:---|
| 正北（下） | 坎 | ☵ | 水 | 陷中有险，但水流不腐，险里藏着生机 |
| 正南（上） | 离 | ☲ | 火 | 光明依附柔顺，如太阳挂在天上 |
| 正东（左） | 震 | ☳ | 雷 | 第一声春雷，惊醒了蛰伏的万物 |
| 正西（右） | 兑 | ☱ | 泽 | 柔在外、刚在内，像湖面映着月亮 |
| 东南（左上）| 巽 | ☴ | 风 | 无孔不入，柔柔地吹，却能穿石 |
| 西南（右上）| 坤 | ☷ | 地 | 厚德载物，包容一切，从不争抢 |
| 东北（左下）| 艮 | ☶ | 山 | 知道该停就停，安静里看见真相 |
| 西北（右下）| 乾 | ☰ | 天 | 刚健不息，靠自己站起来 |

### 阴阳鱼

图中间的太极阴阳鱼，黑里有个白点，白里有个黑点——**"阴中有阳，阳中有阴"**。这是在说：最难的时候，也藏着一丝转机；最顺的时候，也别得意忘形。这世上，本就没有绝对的好事和坏事。

## 🎐 视觉

整个画面用的是**古画宣纸**的感觉，简简单单，不争不抢：

- 背景：宣纸暖灰，像旧书的扉页
- 卦象：赭石色，像古印泥的颜色
- 外圈：古铜色，像老铜镜的边
- **坎卦（☵）和鱼眼**：石青色，全图唯一偏冷的颜色——这是"水"，清凉、安静

没有阴影，没有光效，没有渐变。干干净净，像印在纸上。

## 🖐️ 怎么用

| 怎么操作 | 会怎样 | 什么时候用 |
|------|------|------|
| 点一下 | 慢慢转起来。再点一下，停下来 | 日常烦躁，盯着看，让心静下来 |
| 快速点两下 | 飞快地转。再点两下，停下来 | 脑子里念头太多，让它们被"甩出去" |
| 按住不放 | 越按转得越快，没有上限。松手就保持那个速度 | 特别憋闷的时候，想象烦闷被甩出消散 |
| Ctrl + 滚鼠标 | 放大缩小（40% ~ 150%） | 调到看着最舒服的大小 |
| 按住拖 | 把它挪到你喜欢的位置 | 放屏幕角落，默默陪着你 |
| 右键 | 关掉 | — |

## 🧘 使用心法

1. 打开它，先什么也不做，静静看几秒
2. 点一下，让它转起来
3. 心里轻轻念一句：**"万物都在转，这股烦闷也一样，不会停留"**
4. 深呼吸，眼睛跟着它走，心就定了

## 🔧 装到自己的电脑上

先装好 [Node.js](https://nodejs.org)（18 以上版本就行），然后：

```bash
git clone https://github.com/BIGGG-BOT/bagua-widget.git
cd bagua-widget
npm install
node_modules\electron\dist\electron.exe .
```

Windows 的话，直接双击 `启动八卦.bat` 就行。

> 国内的网络偶尔下不动 Electron？先跑这行再装：`set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/`

## 🛠 按自己的喜好改

打开 `renderer.js`，改这几个数：

| 常量 | 是干什么的 | 默认 |
|------|------|------|
| `currentScale` | 默认大小 | `0.75` |
| `SPEED_SLOW` | 慢转速度（度/秒） | `30` |
| `SPEED_FAST` | 快转速度（度/秒） | `180` |
| `CHARGE_RATE` | 长按加速率 | `120` |

想换配色？直接搜颜色代码（比如 `#F2EFE9`）替换就行。

## 💡 小问题

**Electron 下不动？**

```bash
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install
```

**macOS 下窗口有白边？**

在 `main.js` 里加一行：

```js
mainWindow.setHasShadow(false);
```

**想开机就自己出来？**

把 `启动八卦.bat` 的快捷方式扔到 Windows 启动文件夹（`Win+R` → `shell:startup`）。
